Class Sequence.sequence

Given a list of sources, play one track from each of the first * sources, then loop on the last one. Optionally, merge tracks when * advancing in the sequence. The merge flag will *not* merge tracks * while looping on the last source -- this behavior would not be suited * to the current use of sequence in transitions.
